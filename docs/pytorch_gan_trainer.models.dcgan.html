train(epochs, dataloader, epoch_start=0, output_batch=64, output_epochs=1, output_path='./outputs', project=None, name=None, config={}, models_path=None)¶

Training loop for DCGAN.

Parameters

  • epochs (int) – Number of epochs for training.

  • dataloader (DataLoader) – PyTorch DataLoader containing the dataset.

  • epoch_start (int) – The epoch from which training should start.

  • output_batch (int) – The batch size for the outputs.

  • output_epochs (int) – The frequency for which outputs will be generated (per epoch).

  • name (str) – Experiment name (Weights and Biases only).

  • config (dict) – Dictionary containing the configuration settings.

  • models_path (str) – Path at which (if provided) the checkpoints will be saved.
